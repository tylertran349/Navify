Information

This is a class project for the course ECS170 Introduction to AI at UC Davis. It calculates the shortest path between or within two buildings and takes into account various factors, including:

  • Weather
  • Traffic
  • Transportation mode
  • Distance from crime areas

The project uses computer vision to analyze floorplans from different buildings and implements an A* algorithm to find the shortest path within and between buildings. Filtering for the pathfinding algorithm is based on various APIs and open-source libraries.

Details of the implementation are available on GitHub.
